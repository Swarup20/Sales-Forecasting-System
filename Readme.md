# 📈 Sales Forecasting System

## 📌 Project Overview

This project was developed as part of the **Xylofy AI Internship**. It predicts future product demand using historical retail sales data and provides an interactive dashboard for inventory planning and business decision-making.

---

## 🎯 Objectives

- Forecast future sales using multiple forecasting models
- Detect unusual sales patterns using anomaly detection
- Segment products based on demand behavior
- Build an interactive Streamlit dashboard

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Scikit-learn
- Statsmodels
- Prophet
- XGBoost
- Streamlit
